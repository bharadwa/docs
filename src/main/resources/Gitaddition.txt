Generate an SSH key (if you don't have one):</br>
Open Command Prompt or Git Bash and run: </br>
ssh-keygen -t ed25519 -C "your_email@example.com"</br>
Press Enter to accept the default file location.</br>
Add your SSH key to the ssh-agent:</br>
eval "$(ssh-agent -s)"</br>
navigate to users/Administrator/.ssh and run this command</br>
ssh-add %USERPROFILE%\.ssh\id_ed25519</br>
Copy your public key:</br>
type %USERPROFILE%\.ssh\id_ed25519.pub</br>
Copy the output.</br>
Go to GitHub > Settings > SSH and GPG keys > New SSH key. Paste your key and save.</br>
Test your connection:</br>
ssh -T git@github.com</br>
